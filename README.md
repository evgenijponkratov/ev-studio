# EV Studio

EV Studio — учебный лендинг веб-студии, созданный на HTML, CSS и JavaScript.

## Описание проекта

Сайт демонстрирует услуги веб-студии: создание лендингов, интернет-магазинов и поддержку сайтов.

## Что реализовано

- Адаптивная верстка
- Бургер-меню для телефона
- Плавная прокрутка по разделам
- Форма заявки с проверкой полей
- Анимация появления блоков при прокрутке
- Публикация через GitHub Pages

## Технологии

- HTML
- CSS
- JavaScript
- GitHub Pages
